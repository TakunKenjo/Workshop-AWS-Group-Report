Before starting the deployment of the **SmartDocs AI** system on the AWS platform, the **team** carried out several necessary preparation steps to ensure that the deployment environment and required resources were fully set up. These steps included preparing the project source code, preparing an AWS account, and creating an IAM User to manage access permissions to AWS services.

Proper preparation of these components allows the team to carry out the subsequent deployment steps smoothly while ensuring that AWS resources and access permissions are managed appropriately.
