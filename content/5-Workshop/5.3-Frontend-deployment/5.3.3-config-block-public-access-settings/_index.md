### 1. In the S3 Bucket Interface

* Select the ***Permissions*** tab.
* You will see that ***Block all public access*** is currently set to ***On***.
* Under **Block public access (bucket settings)**, select ***Edit***.

![permission](/images/5-Workshop/5.3-Frontend-deployment/5.3.3-config-block-public-access-settings/permission.png)

### 2. In the Block Public Access Interface

* Uncheck ***Block all public access***.
* Then, select ***Save changes***.

![edit](/images/5-Workshop/5.3-Frontend-deployment/5.3.3-config-block-public-access-settings/edit.png)

* To confirm the changes, enter `confirm` in the input field and select ***Confirm***.

![confirm](/images/5-Workshop/5.3-Frontend-deployment/5.3.3-config-block-public-access-settings/confirm.png)

### 3. Verify the Block Public Access Configuration

* ***Block all public access***: The status is displayed as ***Off***.

![check](/images/5-Workshop/5.3-Frontend-deployment/5.3.3-config-block-public-access-settings/check.png)
